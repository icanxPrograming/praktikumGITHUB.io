Panduan Dasar Git & GitHub

Belajar sistem version control dengan mudah dan terstruktur.

1. Apa itu Git?

Git adalah sistem version control yang digunakan untuk mencatat perubahan kode, memudahkan kolaborasi tim, dan mengelola riwayat pengembangan project. Dengan Git, kita bisa melacak perubahan, kembali ke versi sebelumnya, dan bekerja secara paralel tanpa takut kehilangan progress.

2. Apa itu GitHub?

GitHub adalah platform hosting repository berbasis Git. Dengan GitHub, kamu bisa menyimpan project-mu di cloud, berkolaborasi dengan programmer lain, melakukan pull request, review code, dan manajemen issue.

3. Perbedaan Git dan GitHub

Git adalah tools version control berbasis terminal/command line untuk menyimpan riwayat perubahan kode di lokal. Sedangkan GitHub adalah layanan cloud untuk menyimpan repository Git secara online, serta mendukung kolaborasi tim.
